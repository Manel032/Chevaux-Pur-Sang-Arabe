# Chevaux Pur Sang Arabes - Application Web

## Description

Cette application web permet de gérer une base de données de chevaux pur sang arabes en Tunisie, ainsi que leurs propriétaires (owners), jockeys et courses.  
Le projet est composé d'un **backend PHP (REST API)** et d'un **frontend React**.

---

## Fonctionnalités

### Chevaux
- Ajouter, modifier et supprimer des chevaux.
- Afficher la liste des chevaux avec informations (nom, âge, owner, jockey).

### Owners
- Ajouter, modifier et supprimer des propriétaires.
- Afficher la liste des propriétaires avec leurs informations.

### Jockeys
- Ajouter, modifier et supprimer des jockeys.
- Afficher la liste des jockeys avec leurs informations.

### Courses
- Ajouter, modifier et supprimer des courses.
- Afficher la liste des courses avec les chevaux associés.

---

## Stack Technique

### Backend
- PHP 8.x
- MySQL / MariaDB
- Architecture MVC
- API REST JSON
- XAMPP / LAMP pour serveur local

### Frontend
- React 18
- Axios pour les requêtes API
- Bootstrap 5 pour le design rapide
- Structure modulaire par composants
